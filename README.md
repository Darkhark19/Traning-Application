# Traning-Application
Application simulating a management center of resources 
## A instalar:

- Intalar o node
- Instalar o yarn ($npm install --global yarn)

### Correr o backend

- $cd src
- $yarn install
- $yarn dev
  Após este último o servidor abre na porta 4000

### Correr o Frontend

- $cd template
- $yarn install
- $yarn dev

### Correr db
- yarn prisma migrate dev
